No CMD:
sudo apt update && sudo apt upgrade -y

Instalando o Apache2:
sudo apt install apache2 -y

Acesse a seguinte pasta:
cd /var/www/html
ls -al
Verifique que terá um arquivo index.html
hostname -I
Teste em seu navegador o IP exibido (http://192.168.1.86)

instalando o PHP:
sudo apt install php -y
sudo rm index.html
sudo nano index.php
<?php echo "hello world"; ?>
Ctrl+X, Y, ENTER

Restart no Apache2:
sudo service apache2 restart

Acesse o IP mais uma vez.

Remova o arquivo index.php:
sudo rm index.php

Instalando Mariadb:
sudo apt install mariadb-server php-mysql -y
sudo service apache2 restart

Configurando mariadb:
sudo mysql_secure_installation
Insira a sua senha para o usuário root, enter, y, enter, digite nova senha, enter, y, y, y, y

instalando o phpmyadmin:
sudo apt install phpmyadmin -y

PHPMyAdmin installation program will ask you few questions. We’ll use the dbconfig-common.
Select Apache2 when prompted and press the Enter key
Configuring phpmyadmin? OK
Configure database for phpmyadmin with dbconfig-common? Yes
Type your password and press OK

Habilitando o PHP MySQLI
sudo phpenmod mysqli

Restart no apacher2
sudo service apache2 restart

modificando a pasta:
sudo ln -s /usr/share/phpmyadmin /var/www/html/phpmyadmin

acessando o phpmyadmin
http://192.168.1.86/phpmyadmin

digite root como usuário e use a senha que você criou

config para webpages:
ls -lh /var/www/
sudo chown -R pi:www-data /var/www/html/
sudo chmod -R 770 /var/www/html/
ls -lh /var/www/
sudo phpenmod mysqli
